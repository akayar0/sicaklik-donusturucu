🌡️ Sıcaklık Dönüştürücü (Temperature Converter)
Bu proje, Celcius ve Fahrenheit değerlerini gerçek zamanlı olarak birbirine dönüştüren, modern arayüzlü bir web uygulamasıdır. Kullanıcı deneyimini ön planda tutan animasyonlar ve mobil uyumlu (responsive) bir yapı ile geliştirilmiştir.

🚀 Özellikler
Gerçek Zamanlı Dönüşüm: oninput event'i ile değer girildiği anda hesaplama yapılır.

Modern Tasarım: Glassmorphism etkileri, CSS transistion ve hover animasyonları ile zenginleştirilmiş kullanıcı arayüzü.

Tam Uyumluluk: Media Queries kullanılarak tüm ekran boyutlarına (Mobil, Tablet, Masaüstü) tam uyumlu hale getirilmiştir.

Hassas Hesaplama: toFixed(2) fonksiyonu ile sonuçlar virgülden sonra iki basamak olacak şekilde optimize edilmiştir.

🛠️ Kullanılan Teknolojiler
HTML5: Semantik yapı ve form kontrolleri.

CSS3: Flexbox, Responsive Design, CSS Nesting ve Gelişmiş Animasyonlar.

JavaScript (Vanilla JS): Dinamik veri manipülasyonu ve matematiksel hesaplamalar.
